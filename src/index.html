Prečo tento web

zenova.sk ukazuje celý flow od návrhu webu po prevádzku v Azure

Web nebudujeme len ako marketingovú stránku. Má slúžiť aj ako verejný dôkaz toho, ako vyzerá malý, dobre zdokumentovaný end-to-end delivery flow: od návrhu architektúry a HLD, cez implementáciu vo VS Code a GitHub Actions workflowy, až po nasadenie a prevádzku v Azure.

Návrh a obsah v repozitári

Bez CMS, bez databázy a bez zbytočných prevádzkových závislostí. Obsah, štruktúra a rozhodnutia sú verzované v repozitári a pripravené na čistý review proces.

Automatizovaný delivery flow

Lokálny vývoj vo VS Code, validácia v repozitári a GitHub Actions workflowy vytvárajú čitateľný tok od zmeny v kóde po kontrolovaný release.

Azure prevádzka s IaC smerom

Hosting na Static Web Apps, infra smer v Bicep a samostatné test/prod workflowy ukazujú, ako vyzerá rozumný Azure základ bez zbytočného platformového overkillu.
