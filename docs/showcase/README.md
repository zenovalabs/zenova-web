# Use Case Page Templates

This folder contains showcase-oriented templates and reference snippets for future use-case pages on `zenova.sk`.

The current baseline pattern comes from the working IoT case and is intended to keep future pages visually aligned with the rest of the website.

## Recommended page structure

Use this order for new detail pages:

1. `page-hero`
2. `meta-grid` overview
3. `metric-showcase-grid` impact highlights
4. `split-grid` with `detail-card` sections for context, solution, impact, outcome
5. `cta-band` closing section

## Shared CSS building blocks

These classes already exist in the public website stylesheet:

- `meta-grid` and `meta-item`
- `metric-showcase-grid`, `metric-showcase-card`, `metric-showcase-value`
- `split-grid` and `detail-card`
- `detail-list`
- `cta-band` and `cta-actions`

## Authoring guidance

- Keep the top `meta-grid` factual and compact
- Use the `metric-showcase-grid` only for highlights that are defensible
- Prefer 4 metric cards for visual balance
- Keep the card title short and the supporting text to 1-2 sentences
- If a number is not proven, use a plain value label instead of inventing a percentage
- Reuse the same section order in SK and EN versions
